## What is MediaPollingBean?

MediaPollingBean is a custom **Java Bean component** that provides enterprise-level cloud integration capabilities for Tubify. It enables secure authentication, real-time synchronization, and seamless media management between local storage and cloud services.

This external dependency is a reusable JavaBean that can be integrated into any Java Swing application via the NetBeans GUI Builder or programmatically.

<Note>
  MediaPollingBean is an open-source component developed separately from Tubify.
  View the source code on [GitHub](https://github.com/Perelluent/MediaPollingBean).
</Note>

## How Tubify Integrates MediaPollingBean

Tubify integrates the component in several key areas:

### 1. Initialization in MainWindow

The MediaPollingBean is instantiated and configured when the application starts:

```java MainWindow.java
public class MainWindow extends JFrame implements MediaPollingBeanListener {
    private final MediaPollingBean mediaPollingBean;
    
    public MainWindow() {
        // Initialize and configure the bean
        mediaPollingBean = new MediaPollingBean();
        mediaPollingBean.setApiUrl("https://difreenet9.azurewebsites.net");
        mediaPollingBean.setPollingInterval(30);
        
        // Register as event listener
        mediaPollingBean.addMediaPollingBeanListener(this);
    }
    
    @Override
    public void onNewMediaFound(MediaPollingBeanEvent evt) {
        // Reload library when new media is detected
        if (libraryPanel != null) {
            libraryPanel.loadMedia();
        }
    }
}
```

### 2. Authentication via LoginPanel

The LoginPanel uses MediaPollingBean to authenticate users:

```java LoginPanel.java
private void btnLoginActionPerformed(ActionEvent evt) throws Exception {
    String email = txtUser.getText();
    var password = new String(pswPassword.getPassword());
    
    try {
        String token = mediaPollingBean.login(email, password);
        mainWindow.setToken(token);
        mainWindow.showMainWindow();
    } catch (Exception e) {
        lblError.setText("Login failed. Please check your username and password.");
    }
}
```

**Reference:** [LoginPanel.java:186-210](/home/daytona/workspace/source/src/main/java/es/perelluent/tubify/LoginPanel.java:186)

### 3. Media Management in LibraryPanel

The LibraryPanel uses MediaPollingBean to upload, download, and retrieve media:

```java LibraryPanel.java
// Upload a local file to the cloud
main.getMediaPollingBean().uploadFileMultipart(fileToUpload, "");

// Download a cloud file
main.getMediaPollingBean().download(media.id, tempFile);

// Get all media from the cloud
List<Media> cloudList = main.getMediaPollingBean().getAllMedia();
```
