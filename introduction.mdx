# Welcome to Tubify

Tubify is a desktop application built with **Java Swing** that acts as a **GUI wrapper** around the powerful command-line tool [`yt-dlp`](https://github.com/yt-dlp/yt-dlp). It allows users to download videos or audio files easily from multiple platforms without technical complexity.

## What is Tubify?

Tubify transforms the command-line experience of yt-dlp into an intuitive, modern desktop application. Whether you want to download videos, extract audio, or manage your media library, Tubify provides a seamless experience with cloud synchronization capabilities.

## Key Features

### Download Management

<Steps>
  <Step title="Multiple Format Support">
    Download videos in various resolutions (1080p, 720p, 480p, or best available) or extract audio in MP3, M4A, or WAV formats.
  </Step>
  <Step title="Real-time Progress Tracking">
    Monitor download progress with a visual progress bar that updates in real-time, showing percentage completion.
  </Step>
  <Step title="Smart URL Validation">
    Built-in URL validation ensures you're downloading from valid sources before processing begins.
  </Step>
</Steps>

### User Interface

- **Modern Design**: Utilizes FlatLaf library for a clean, contemporary aesthetic with a high-contrast color palette (dark grey background with pink accents using brand color `#fb3f62`)
- **Theme Toggle**: Seamlessly switch between Dark and Light modes to suit your preference
- **Responsive Layouts**: Uses BorderLayout and MigLayout for dynamic resizing that maintains visual integrity when maximized
- **Custom Typography**: Montserrat font with different weights (Bold for titles, Regular for data) for improved legibility
- **Interactive Icons**: Buttons display as primary action components that transition into icons on mouse hover with descriptive tooltips

### Cloud Integration

Powered by [MediaPollingBean](https://github.com/Perelluent/MediaPollingBean), Tubify provides enterprise-level cloud features:

<Note>
  **Secure Authentication**: User login with JWT token persistence and "Remember Me" functionality ensures your sessions are secure and convenient.
</Note>

- **Real-Time Polling**: The app automatically listens for changes on the server (every 30 seconds by default) and updates the UI instantly when new files are added
- **Upload & Download**: Upload local media to the cloud or stream cloud media directly
- **Smart Playback**: Plays local files instantly with your system's default media player, or streams cloud files by downloading them temporarily

### Library Management

- **Unified View**: See both local files and cloud media in a single table with visual indicators
- **Real-time Search**: Filter your media library using regular expressions as you type
- **Auto-Selection**: After a successful download, the system automatically scrolls to and selects the new file
- **File Operations**: Play, upload, or delete media files directly from the library interface

### Performance & Concurrency

<Warning>
  All heavy processes use SwingWorker for multi-threading, ensuring the UI remains responsive (non-blocking) while downloads or cloud sync operations are in progress.
</Warning>

- **Background Processing**: Video downloading via yt-dlp and cloud synchronization run in background threads
- **State Control**: Download button is disabled during active processes to prevent bandwidth saturation or file conflicts
- **Informative Feedback**: Status reporting with color-coded labels (success or red for failure) keeps you informed
