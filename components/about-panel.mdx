## Overview

`AboutPanel` displays information about the Tubify application, including author details, used resources, repository link, and the application logo. It provides a centralized location for users to learn about the application and its dependencies.

## Layout

The panel uses MigLayout with a card-style design:

```java AboutPanel.java:51-58
JPanel pnlAbout = new JPanel(new MigLayout("wrap, insets 40, gapy 12", "[grow, fill]"));

pnlAbout.setOpaque(true);
pnlAbout.putClientProperty("FlatLaf.style", "arc: 25");       
pnlAbout.setBorder(new CompoundBorder(
    new FlatLineBorder(new Insets(0,0,0,0), Color.GRAY, 1, 25),
    new EmptyBorder(10, 10, 10, 10)
));
```

Features:
- Rounded corners with 25px arc radius
- Gray border with padding
- Vertical layout with 12px gaps between components
- 40px insets for spacing

## Navigation

### Back Button

```java AboutPanel.java:96-105
JButton btnBack = new JButton("Back to App");

btnBack.putClientProperty("FlatLaf.style", "arc: 15; background: #fb3f62; foreground: #ffffff");

btnBack.addActionListener(new ActionListener() {
    @Override
    public void actionPerformed(ActionEvent e) {
        main.showMainWindow();
    }
});
```

The "Back to App" button allows users to return to the main application window. It features:
- Pink accent color (#fb3f62)
- Rounded corners (15px arc)
- Hand cursor on hover
- Returns to main window on click

## Accessing the About Panel

The About panel is accessible from the main menu:

```java MainWindow.java
mniAbout = new JMenuItem("About");
mniAbout.addActionListener(e -> showAbout());
mnuHelp.add(mniAbout);
```

Navigate to **Help → About** in the menu bar to display this panel.

## Styling

The AboutPanel uses the Montserrat font family throughout:
- **Bold, 14pt** - Section titles
- **Plain, 16pt** - Author name
- **Plain, 14pt** - Course info
- **Plain, 13pt** - Resources list
- **Plain, 12pt** - Repository URL

The panel integrates with the FlatLaf theme system and respects the application's dark/light mode settings.
