## Overview

`LoginPanel` handles user authentication within Tubify. It provides a modern login interface with email/password fields, "Remember Me" functionality using Java Preferences, and automatic session restoration.

</ParamField>

<ParamField body="Extends" type="class">

</ParamField>

## Authentication Flow

### btnLoginActionPerformed()

Handles the login process with the following steps:

1. Retrieves email and password from input fields
2. Calls `mediaPollingBean.login()` to authenticate
3. Stores or clears credentials based on "Remember Me" state
4. Updates the main window with the JWT token
5. Navigates to the main window on success
6. Displays error message on failure

<Accordion title="Login Implementation">
```java
private void btnLoginActionPerformed(ActionEvent evt) throws Exception {
    String email = txtUser.getText();
    var password = new String(pswPassword.getPassword());
    lblError.setText("");
    
    try {
        String token = mediaPollingBean.login(email, password);
        mainWindow.setToken(token);

        if (chkRememberMe.isSelected()) {
            prefs.put(prefEmail, email);
            prefs.put(prefToken, token);
            prefs.putBoolean(prefRememberMe, true);
        } else {
            // If unchecked, remove stored credentials
            prefs.remove(prefEmail);
            prefs.remove(prefToken);
            prefs.putBoolean(prefRememberMe, false);
        }

        mainWindow.showMainWindow();
    } catch (Exception e) {
        lblError.setText("Login failed. Please check your username and password.");
        e.printStackTrace();
    }
}
```
</Accordion>

## Automatic Login

### checkRememberMe()

Called during application startup to restore previous sessions:

<Accordion title="Auto-Login Implementation">
```java
public void checkRememberMe() {
    boolean remember = prefs.getBoolean(prefRememberMe, false);
    if (remember) {
        String savedEmail = prefs.get(prefEmail, "");
        String savedToken = prefs.get(prefToken, "");

        if (!savedToken.isEmpty()) {
            try {
                mediaPollingBean.setToken(savedToken);
                mainWindow.setToken(savedToken);
                mainWindow.showMainWindow();
                return;
            } catch (Exception e) {
                prefs.remove(prefToken);
                lblError.setText("Session expired. Please login again.");
                System.out.println("Token validation failed: " + e.getMessage());
            }
        }
        txtUser.setText(savedEmail);
        chkRememberMe.setSelected(true);
    }
}
```
</Accordion>

This method is called from `MainWindow` constructor:

```java
// In MainWindow constructor
loginPanel.checkRememberMe();
```

## Session Management

### clearTextAreas()

Clears credentials and resets the form (called during logout):

```java
public void clearTextAreas() {
    prefs.remove(prefToken);
    prefs.remove(prefEmail);
    txtUser.setText("");
    pswPassword.setText("");
}
```

## JWT Token Flow

1. User enters credentials and clicks "LOGIN"
2. `mediaPollingBean.login(email, password)` sends credentials to backend
3. Backend validates and returns JWT token
4. Token is stored in:
   - `MainWindow.token` (runtime state)
   - `MediaPollingBean.token` (for API calls)
   - `Preferences` (persistent storage if "Remember Me" is checked)
5. Token is used for all subsequent API requests
6. On logout, token is cleared from all locations

## Styling with FlatLaf

The panel uses FlatLaf properties for modern appearance:

- **Rounded card**: `arc: 30` on the form panel
- **Rounded inputs**: `arc: 15` on text fields
- **Rounded button**: `buttonType: roundRect` on login button
- **Custom border**: `FlatLineBorder` with rounded corners

## Integration with MediaPollingBean

After successful authentication:

```java
mediaPollingBean.setToken(token);
mediaPollingBean.setRunning(true); // Starts polling for media updates
```

This enables:
- Automatic synchronization of cloud media
- Periodic polling for new media items
- Upload/download functionality in LibraryPanel
