## Overview

Tubify is a modern desktop application that provides a user-friendly graphical interface for the powerful yt-dlp command-line tool. Download videos and audio from multiple platforms with ease, manage your media library, and sync files to the cloud.

## Quick Start

<Steps>
  <Step title="Install Dependencies">
    Download and install yt-dlp and optionally FFmpeg for audio extraction
    
    ```bash
    # Download yt-dlp from GitHub releases
    https://github.com/yt-dlp/yt-dlp/releases
    
    # Download FFmpeg (optional, for audio extraction)
    https://ffmpeg.org/download.html
    ```
  </Step>
  
  <Step title="Build and Run">
    Build the project with Maven and launch the application
    
    ```bash
    mvn clean package
    java -jar target/Tubify-1.0-SNAPSHOT.jar
    ```
  </Step>
  
  <Step title="Configure Settings">
    Set your yt-dlp binary path and download folder in Preferences
  </Step>
  
  <Step title="Start Downloading">
    Paste a video URL and click Download to save media to your library
  </Step>
</Steps>

## Technology Stack

Tubify is built with modern Java technologies:

- **Java Swing** - Desktop GUI framework (JDK 24)
- **FlatLaf** - Modern look and feel library
- **MigLayout** - Flexible layout manager for responsive UI
- **MediaPollingBean** - Custom Java Bean for cloud integration
- **yt-dlp** - Powerful media downloader
- **FFmpeg** - Audio/video processing

## Architecture Highlights

- **Multi-threading with SwingWorker** - Non-blocking UI during downloads
- **Real-time polling** - Automatic cloud sync with server updates
- **Regex-based search** - Dynamic table filtering as you type
- **JWT authentication** - Secure cloud access with token persistence
- **Event-driven design** - Responsive UI with custom event listeners
